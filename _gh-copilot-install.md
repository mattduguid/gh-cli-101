# gh-cli-install

## main

- https://cli.github.com/

## install

- https://github.com/cli/cli?tab=readme-ov-file#installation

## use

```bash
# login
gh auth login --hostname github.com OR <DOMAIN>.ghe.com

# logout
gh auth logout -h github.com OR <DOMAIN>.ghe.com -u <USER>

# status
gh auth status
```
